# 🧠 AI SYSTEM INSTRUCTIONS FOR RTP-CORE
1. **Teknoloji:** Rust 1.84+, C (FFI: bcg729).
2. **İş Mantığı Yasağı:** Bu kütüphane "hangi codec seçilmeli" veya "paket nereye yönlendirilmeli" gibi kararlar (Dialplan/Logic) veremez. Sadece RTP başlıklarını, Jitter Buffer'ı ve DSP (Cubic Resampling, Anti-Aliasing) işlemlerini saf matematikle yürütür.
3. **Ağ (I/O) Yasağı:** UDP soket açma veya dinleme işlemleri burada YASAKTIR. Sadece kendisine dışarıdan verilen byte array'leri işler.
4. **Performans (Zero-Cost):** `unwrap()` kullanımından kaçınılmalı, ses döngüsü (Audio Loop) içinde ağır bellek tahsisatı (allocation) veya mutex kilitlerinden (lock contention) kaçınılmalıdır.
